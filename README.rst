soapy_power
===========

Obtain power spectrum from SoapySDR devices (RTL-SDR, Airspy, SDRplay, HackRF, bladeRF, USRP, LimeSDR, etc.)

Requirements
------------

- Python 3
- simplesoapy (https://github.com/xmikos/simplesoapy)
